# kaczynski-ted
"Kaczynski" Text EDitoris is a modal text editor inspired by Vim, Nano and others.

## Keybinds

### NORMAL mode
- arrow keys => move cursor
- [enter] => write mode
- v => select mode (not implemented yet)
- m => move mode (not implemented yet)
- $ => shell mode
- Q => quit
- s => save
- S => save as
- X => save & quit
- d/[delete] => delete character at cursor
- D => delete line
- x => select line (not implemented yet)
- T => go to top
- B => go to bottom
- [pgup] => scroll up one screen
- [pgdn] => scroll down one screen

### NORMAL mode
- arrow keys => move cursor
- [escape] => normal mode
- [backspace] => delete character before cursor
- [delete] => delete character at cursor
- [enter] => new line
